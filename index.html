About

rzoom v2 is a pure and simple JavaScript image magnifier. You can extend it and modify easily. It supports modern browsers including IE7. v2 version is a bit different than the first one. Well, see for yourself!

Example

Features

Pure JavaScript
Sometimes you can’t use a widget written in jQuery or other libraries. rzoom will come in handy in such situations.
Automatic detection
If a scaled image is smaller than its original, rzoom is automatically disabled.
Supports many browsers
When I tested it in January it worked even in IE7. But IE7 is not relevant these days
Respects ‘auto’ CSS value
rzoom respects CSS auto value of width or hight attributes if you want to preserve the aspect ratio.
2 ways of initiating
You can use the add method either in onload event handler or on its own.

Options

You can pass a few options like:

var magnifier = magnifier({
	parentId: "photoContainer",
	width: 400,
	height: 400,
	offset: 10
});
parentId (required)
The container the widget is attached to. parentId is an id of an element.
width (optional)
width of the widget; defaults to 300
height (optional)
height of the widget; defaults to 300
offset (optional)
margin between the container and the widget; defaults to 0

How To Use

Load the widget first with at least parentId option: 

<script type="text/javascript" src="widget/lib/magnifier.js"></script>
<script type="text/javascript">
	var magnifier = magnifier({parentId: "photoContainer"});
</script>

Initiate it on an image: 

<img src="01.jpg" style="width: 300px; height: auto" onload="magnifier.add( this );" />

You don’t need to use add method in the onload event handler. You can use it anywhere. Just pass it an IMG DOM element.
